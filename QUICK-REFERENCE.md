## ğŸ“š QUICK GRAMMAR REFERENCE

### Tenses Quick Reference

| Tense | Formula | Usage |
|-------|----------|--------|
| Present Simple | V1/V1+s/es | Facts, habits |
| Present Continuous | am/is/are + V-ing | Happening now |
| Present Perfect | have/has + V3 | Unspecified past, experiences |
| Present Perfect Continuous | have/has + been + V-ing | Started past, still continuing |
| Past Simple | V2 | Completed, specific time |
| Past Continuous | was/were + V-ing | Was happening at past time |
| Past Perfect | had + V3 | Before another past action |
| Past Perfect Continuous | had been + V-ing | Continuing until past action |
| Future Simple | will + V1 | Predictions, decisions |
| Future Continuous | will be + V-ing | In progress at future time |
| Future Perfect | will have + V3 | Completed before future time |

---

### Subject-Verb Agreement

| Subject | Verb |
|---------|-------|
| Singular | Singular (+s/es) |
| Plural | Plural (no -s/es) |
| Each/Every | Singular |
| Either/Neither | Singular (matches nearest) |
| Collective Noun | Singular (unit) or Plural (individuals) |
| A number of | Plural |
| The number of | Singular |

---

### Conditionals Quick Reference

| Type | Structure | Usage |
|------|-----------|--------|
| 0 | If + Present, Present | General truths |
| 1 | If + Present, will + Base | Real possibilities |
| 2 | If + Past, would + Base | Unreal/hypothetical |
| 3 | If + Past Perfect, would have + V3 | Unreal past |
| Mixed | Various | Past condition/present result |

---

### Passive Voice Quick Reference

| Tense | Active | Passive |
|-------|--------|--------|
| Present Simple | I write | It is written |
| Present Continuous | I am writing | It is being written |
| Present Perfect | I have written | It has been written |
| Past Simple | I wrote | It was written |
| Past Continuous | I was writing | It was being written |
| Past Perfect | I had written | It had been written |
| Future Simple | I will write | It will be written |
| Modals | I can write | It can be written |
